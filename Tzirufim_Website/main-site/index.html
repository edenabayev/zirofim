03 — מרצים

האנשים שמובילים

ד״ר רועי יוזביץ׳

מחשבה ותודעה

לפרק את המנגנון המחשבתי ולגעת בנשמה.

ניכנס לעומקן של האשליות הקוגניטיביות שמנהלות אותנו ביומיום, נלמד כיצד לקלף תבניות מקובעות ונחווה רגע בו המחיצות קורסות ונשאר קיום חשוף, נקי ואמיתי.

ד״ר דודי אדלר

נשימה אקסטטית

מסע פנימי ועוצמתי אל מעמקי הלא־מודע דרך "נשימת האחד".

בשילוב ריברסינג וקבלה, נפרמט את הגוף, נשחרר חסמים כבדים ונגיע בלקיחת אוויר אחת להתעלות נפשית ושמחה עמוקה.

רב אלישע וייל

דבקות ותפילה

חיבור בלתי אמצעי המחבר בין תנועה, מילים והשתיקה שביניהן.

יחד ניגע בסוד הדבקות החסידית, נלמד כיצד למסור את עצמנו במלוא הכוונה, ולהפוך את הגוף לכלי טהור שמוצף באור והתרוממות רוח.

נועם אלמוג

ניגון וקול פנימי

הזמנה לגלות את העוצמה המרפאת של הקול האנושי החשוף.

לא רק להאזין, אלא ללמוד לייצר תדר שמרעיד את חדרי הלב, ולהפוך את הקול לתיבת התהודה הטבעית של הנפש והמרחב.
